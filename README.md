# Globics API System

-   Node version: 12.x
-   MySQL 8.0.20

# Prepare working enviroment

-   Please read the docs to configure Prettier for your prefer code editor
-   https://prettier.io/docs/en/editors.html

## Local dev setup

-   Import SQL db from database/sql
-   Change connect info: .env
-   Start MYSQL server
-   Install node modules: $npm install
-   Start a local dev: $npm run dev
